2020

Crop Identification Using Clustering Algorithm and Satellite Images

automated crop identification

This project focuses on automated crop identification using clustering algorithms applied to satellite imagery. The primary goal is to classify different types of crops based on spectral patterns, helping farmers and agricultural departments optimize land use and improve yield predictions. The system uses unsupervised learning techniques like K-Means clustering to segment satellite images into different crop zones. Data preprocessing includes image enhancement, noise reduction, and feature extraction. The identified crop patterns are validated using real-time data from agricultural sources. This project provides valuable insights for precision farming, disaster management, and efficient resource allocation. By automating the crop classification process, it reduces human effort, improves accuracy, and enhances agricultural productivity.

2018

Tampering Avoidance Using Enhanced Random LSB Technique

Digital image tampering using LSB

This project aims to detect and prevent digital image tampering using an enhanced Least Significant Bit (LSB) technique. It ensures the integrity of images by embedding a randomized, secure watermark into the least significant bits of an image. The system encrypts watermark data before embedding, making unauthorized modifications easily detectable. If any pixel is altered, the extraction process detects inconsistencies, flagging potential tampering. This method enhances image security in forensics, medical imaging, and digital signature verification, providing a robust solution against fraudulent image alterations. The approach is efficient, lightweight, and highly secure, making it ideal for real-world applications
